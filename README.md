# 🌌 3D Solar System Explorer

[Live Demo](https://YOURUSERNAME.github.io/solar-system-explorer/)

## Features
- Interactive 3D planets with realistic orbits
- Touch/click on any planet to learn facts
- Smooth camera controls (zoom, rotate, drag)
- Earth & Mars with custom surface textures
- Built with Three.js

## Technologies
- Three.js (WebGL)
- HTML5/CSS3
- JavaScript (ES6)

## How to Use
- Drag to rotate view
- Scroll to zoom in/out
- Tap/click any planet for information
- Use arrow buttons to navigate
